## 🆘 IF SOMETHING GOES WRONG

### SQL Errors:
- **Error:** "relation already exists"
- **Fix:** Tables might exist. Drop them first: `DROP TABLE IF EXISTS customer_churn_analysis CASCADE;`

### Frontend 404:
- **Error:** Page not found
- **Fix:** Hard refresh (Ctrl+Shift+R), clear cache, verify Vercel deployed

### Backend Errors:
- **Error:** API returns 500
- **Fix:** Check Railway logs, verify backend restarted after code push

### No Data Showing:
- **Error:** Empty tables
- **Fix:** Need customer data with visits. Click Refresh to run analysis.

### Wrong Language:
- **Error:** Arabic shows English text
- **Fix:** Check language toggle in top right, reload page
