======================================
   START YOUR APP - 3 SIMPLE STEPS
======================================

✅ Step 1: Setup Supabase (5 minutes) - FREE!
--------------------------------------
1. Go to: https://supabase.com
2. Click "Start your project" 
3. Sign up (it's FREE!)
4. Create new project:
   - Name: loyaltypass
   - Password: (make a strong one, save it!)
   - Region: (choose closest to you)
   - Click "Create new project"
5. Wait 2 minutes...

✅ Step 2: Setup Database (2 minutes)
--------------------------------------
1. In Supabase, click "SQL Editor" (left sidebar)
2. Click "New query"
3. Open file: supabase/schema.sql
4. Copy ALL content
5. Paste in Supabase SQL Editor
6. Click "Run" (bottom right)
7. Wait 10 seconds... Success!

8. Click "New query" again
9. Open file: supabase/policies.sql
10. Copy ALL content
11. Paste in Supabase SQL Editor
12. Click "Run"
13. Success!

✅ Step 3: Get Your Keys (1 minute)
--------------------------------------
1. In Supabase, click "Settings" (bottom left)
2. Click "API"
3. You'll see:
   - Project URL: https://xxxxx.supabase.co
   - anon public key: eyJ... (long string)

4. Open file: frontend/.env
5. Replace:
   VITE_SUPABASE_URL=https://xxxxx.supabase.co
   VITE_SUPABASE_ANON_KEY=eyJ...your_key...

6. Save the file

✅ Step 4: START THE APP! (30 seconds)
--------------------------------------
Run this command:

   npm run dev

You'll see:
   Local: http://localhost:3000/

Open your browser: http://localhost:3000

🎉 YOUR APP IS RUNNING!
